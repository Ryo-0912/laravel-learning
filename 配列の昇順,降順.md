# 配列の昇順

例1
```
<?php
function bubble_sort(&$array){
    $size = count($array);
    for ($i = 0; $i < $size; $i++) {
        for ($j = 0; $j < ($size - 1 - $i); $j++) {
            if ($array[$j + 1] < $array[$j]) {
                $tmp = $array[$j + 1];
                $array[$j + 1] = $array[$j];
                $array[$j] = $tmp;
            }
        }
    }
}

$array = [100, 3, 30];
bubble_sort($array);
echo implode(",", $array).PHP_EOL;
?>
```

各値の遷移

| $i | $j | $size | $size-1-$i | $j < ($size - 1 - $i) |
|-|-|-|-|-|
| 0 | 0 | 3 | 2 | ⚪︎ |
|  | 1 | 3 | 2 | ⚪︎ |
|  | 2 | 3 | 2 | × |
| 1 | 0 | 3 | 1 | ⚪︎ |
|  | 1 | 3 | 1 | × |
|  | 1 | 3 | 1 | × |
| 2 | 0 | 3 | 0 | × |
|  | 0 | 3 | 0 | × |
|  | 0 | 3 | 0 | × |
