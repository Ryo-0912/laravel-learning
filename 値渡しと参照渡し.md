# 値渡し

変数に変数を代入した時に発生

ex) ***引数に変数を渡すとき(関数やforeachなど)***

phpにおいて、値渡しを防ぐためには次のように、変数の前に「&」をつけると参照渡しとして解釈される。

```
<?php
    $ar = [1,2,3,4,5];

    foreach($ar as $a) {
        $a = $a*2;
    }

    print_r($ar);

    foreach($ar as $a) {
        $a*2;
    }

    print_r($ar);
?>

```

```
Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 4
    [4] => 5
)
Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 4
    [4] => 5
)
```

# 参照渡し

変数のアドレスを渡す

```
<?php
    $ar = [1,2,3,4,5];

    foreach($ar as &$a) {
        $a*2;
    }

    print_r($ar);

    foreach($ar as &$a) {
        $a = $a*2;
    }

    print_r($ar);
?>
```

```
Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 4
    [4] => 5
)
Array
(
    [0] => 2
    [1] => 4
    [2] => 6
    [3] => 8
    [4] => 10
)
```
